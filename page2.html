Organization web development
Front end development
As President of GSILab association, I embarked on enhancing its prominence within my engineering school by creating a website. Utilizing Figma for prototyping and Visual Studio Code for HTML/CSS coding with Tailwind, I spotlighted the laser cutters, 3D printers, CNCs, and the fablab's proximity to the school.
Tailwind expedited CSS development, while responsive design ensured cross-device compatibility. User-friendliness and improved user experience were integral to design considerations. The website adheres to WCAG standards for accessibility, embracing inclusivity for visually impaired users. The project's open-source nature on GitHub reinforces transparency and accessibility.
The resulting webpage elegantly presents the essence of the fablab.

Incorporating Blender-rendered visuals enriched the user interface, amplifying the site's visual appeal and comprehensibility.

This project encapsulates my adeptness in designing, coding, and accessibility practices, amplifying the association's exposure and impact.
